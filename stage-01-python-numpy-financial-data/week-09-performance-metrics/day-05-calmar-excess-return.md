## 1. 今日新名词

- **Calmar 比率**：用几何年化收益除以最大回撤幅度，衡量每承受一单位历史峰谷损失获得了多少年化收益。
- **基准**：用于比较策略表现的参照组合，例如宽基指数、行业指数或与策略风险特征接近的组合。
- **主动收益**：策略单期收益减去基准单期收益，表示该期相对基准多赚或少赚了多少百分点。
- **相对净值**：策略累计增长因子除以基准累计增长因子，表示相同初始资金分别投资两者后的财富比率。
- **跟踪误差**：主动收益序列标准差的年化值，衡量策略相对基准偏离的稳定程度。
- **信息比率**：年化主动收益除以年化跟踪误差，衡量每承担一单位主动波动获得了多少主动收益。
- **相对回撤**：相对净值从历史高点向下回落的幅度，描述策略相对优势曾经恶化多少。

# 一、Calmar 比率

## 3. 定义

本课程统一使用：

\[
Calmar
=
\frac{R_{\text{ann}}}{|MDD|}
\]

其中：

- \(R_{\text{ann}}\) 是同一评价区间内的几何年化收益。
- \(MDD\le 0\) 是带符号最大回撤。
- \(|MDD|=-MDD\ge 0\) 是最大回撤幅度。

不能直接用带负号的 \(MDD\) 作分母，否则正收益策略会得到负 Calmar，失去直观含义。

## 4. 逐步计算

给定三期收益：

\[
10\%,\quad -20\%,\quad 25\%
\]

期初净值设为：

\[
V_0=1
\]

逐期计算：

\[
V_1=1\times1.10=1.10
\]

\[
V_2=1.10\times0.80=0.88
\]

\[
V_3=0.88\times1.25=1.10
\]

从峰值 \(1.10\) 到谷底 \(0.88\) 的回撤为：

\[
MDD
=
\frac{0.88}{1.10}-1
\]

\[
MDD=-20\%
\]

若一年包含三个同类周期，则：

\[
R_{\text{ann}}
=
1.10^{3/3}-1
\]

\[
R_{\text{ann}}=10\%
\]

所以：

\[
Calmar
=
\frac{10\%}{20\%}
\]

\[
Calmar=0.5
\]

## 5. 必须保留期初净值

从收益率计算回撤时，财富路径必须是：

```text
期初净值 → 第一期期末净值 → ... → 第 N 期期末净值
```

例如收益率为：

\[
-20\%,\quad 25\%
\]

正确财富路径是：

\[
1.00\rightarrow0.80\rightarrow1.00
\]

第一期最大回撤是 \(-20\%\)。若代码只保存期末净值 \(0.80,1.00\)，就会把 \(0.80\) 错当作起点，漏掉首期亏损。

## 6. 边界解释

净值单调不下降时 \(MDD=0\)，Calmar 分母为零，课程函数返回 `NaN`。年化收益为负且最大回撤幅度为正时，Calmar 为负。Calmar 只使用一次最严重回撤且对样本区间敏感，必须与样本长度和其他指标一起解释。

# 二、基准与主动收益

## 7. 单期主动收益

设策略与基准在第 \(t\) 期的收益分别为 \(r_{p,t}\) 和 \(r_{b,t}\)，主动收益定义为：

\[
a_t=r_{p,t}-r_{b,t}
\]

例如策略收益为 \(1.2\%\)，基准收益为 \(0.8\%\)，则：

\[
a_t=1.2\%-0.8\%
\]

\[
a_t=0.4\%
\]

主动收益是百分点差，回答“这一期策略比基准多赚了多少”。

## 8. 累计收益差

策略和基准的累计增长因子分别为：

\[
G_{p,T}
=
\prod_{t=1}^{T}(1+r_{p,t})
\]

\[
G_{b,T}
=
\prod_{t=1}^{T}(1+r_{b,t})
\]

累计收益分别为：

\[
R_{p,\text{cum}}=G_{p,T}-1
\]

\[
R_{b,\text{cum}}=G_{b,T}-1
\]

累计收益差为：

\[
\Delta R_{\text{cum}}
=
R_{p,\text{cum}}
-
R_{b,\text{cum}}
\]

它的单位是百分点。

## 9. 相对净值收益

相对净值定义为：

\[
W_{\text{rel},T}
=
\frac{G_{p,T}}{G_{b,T}}
\]

相对净值收益：

\[
R_{\text{rel},T}
=
W_{\text{rel},T}-1
\]

代入累计收益：

\[
R_{\text{rel},T}
=
\frac{1+R_{p,\text{cum}}}
{1+R_{b,\text{cum}}}
-1
\]

它回答策略财富相对基准高或低多少比例。

## 10. 为什么累计收益差不等于相对净值收益

由：

\[
1+R_{p,\text{cum}}
=
(1+R_{b,\text{cum}})
(1+R_{\text{rel},T})
\]

展开：

\[
1+R_{p,\text{cum}}
=
1
+
R_{b,\text{cum}}
+
R_{\text{rel},T}
+
R_{b,\text{cum}}R_{\text{rel},T}
\]

两边减去 \(1\)：

\[
R_{p,\text{cum}}
=
R_{b,\text{cum}}
+
R_{\text{rel},T}
+
R_{b,\text{cum}}R_{\text{rel},T}
\]

再减去基准累计收益：

\[
R_{p,\text{cum}}-R_{b,\text{cum}}
=
R_{\text{rel},T}
(1+R_{b,\text{cum}})
\]

因此一般情况下：

\[
\Delta R_{\text{cum}}
\ne
R_{\text{rel},T}
\]

只有当基准累计收益接近零时，两者才可能近似。

## 11. 不能直接连乘主动收益

精确的单期相对增长率是：

\[
q_t
=
\frac{1+r_{p,t}}{1+r_{b,t}}-1
\]

整理分子：

\[
q_t
=
\frac{r_{p,t}-r_{b,t}}
{1+r_{b,t}}
\]

所以：

\[
q_t
=
\frac{a_t}{1+r_{b,t}}
\]

一般情况下 \(q_t\ne a_t\)，因此不能把 \(\prod(1+a_t)-1\) 当作精确相对净值收益。正确做法是连乘 \((1+r_{p,t})/(1+r_{b,t})\)。

# 三、跟踪表现

## 12. 跟踪误差

主动收益的样本标准差为：

\[
s_a
=
\sqrt{
\frac{
\sum_{t=1}^{N}(a_t-\bar a)^2
}{
N-1
}
}
\]

若一年有 \(A\) 个周期，则年化跟踪误差为：

\[
TE_{\text{ann}}
=
s_a\sqrt{A}
\]

跟踪误差越小，表示策略越稳定地贴近基准；它不判断偏离是好是坏。

## 13. 信息比率

主动收益的算术年化近似为：

\[
\mu_{a,\text{ann}}
=
A\bar a
\]

信息比率定义为：

\[
IR
=
\frac{
\mu_{a,\text{ann}}
}{
TE_{\text{ann}}
}
\]

代入并逐步化简：

\[
IR
=
\frac{
A\bar a
}{
s_a\sqrt A
}
\]

因为：

\[
A=\sqrt A\sqrt A
\]

所以：

\[
IR
=
\frac{\bar a}{s_a}\sqrt A
\]

当跟踪误差为零时，课程函数返回 `NaN`，避免除以零。

## 14. 相对回撤

先构造相对净值：

\[
W_{\text{rel},t}
=
\prod_{i=1}^{t}
\frac{1+r_{p,i}}{1+r_{b,i}}
\]

其历史高点为：

\[
H_{\text{rel},t}
=
\max_{0\le s\le t}W_{\text{rel},s}
\]

相对回撤为：

\[
D_{\text{rel},t}
=
\frac{
W_{\text{rel},t}
}{
H_{\text{rel},t}
}
-1
\]

相对最大回撤是 \(\min_t D_{\text{rel},t}\)。它不是“策略最大回撤减去基准最大回撤”，因为两个最大回撤可能发生在不同时间和不同峰值上。
